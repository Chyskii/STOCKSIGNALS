# Stock Signals

Stock Signals is a modern investment intelligence dashboard focused on AI trends, dividend tracking, market signals, watchlists, and long-term investing education.

## Features

- Dashboard-style landing page
- AI and tech watchlist preview
- Newsletter signup section
- Discord community section
- Responsible investing disclaimer

## Tech Stack

- Vite
- React
- CSS
- Lucide React icons
- Netlify-ready deployment

## Netlify Settings

Build command:

```bash
npm run build
```

Publish directory:

```bash
dist
```
